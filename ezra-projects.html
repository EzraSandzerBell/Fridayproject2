Hello World: This page introduces a basic html document with header tags and an unordered list.

Goodbye: This page is almost identical to Hellow World, other than the text content.

Cookie Recipe: This page introduces ordered lists to an html cookie recipe document.

My First Webpage: This page continues the practice of creating simple html docs with a ul tag.

Blog: This page creates a blog format with a float-right tag for navigation, headers, and page breaks for each entry.

Five Star Resort: This is a fun website with multiple pages that incorporates background images, low opacity boxes for content, and a simple navigation bar.

Cupcake: This cupcake website is fairly simple. Again we used a background image and learned how to stretch it across the page.

Boring Lecture: This page creates left and right margins to center the document. We included some new fonts and the lorem ipsum filler text.

Resume: This is a resume outline that uses header tags, bottom-border page breaks, and ul/ol tags to achieve a professional look.

Greatest Places: This page introduces font colors to a simple ordered list with a photo.

Experiment with Boxes: This page creates a box and explores a framed border style with background color.

Cascading Basics: This page altered the blog's header fonts with cascading techniques.

Animal Shelter Site: This page applies the column technique to our blog format and mimics an animal shelter site. We used css to change the font color of significant text.

Week in Review: This page incorporates bootstrap techniques to create a more professional look. It includes container, jumbotron, boxes with headers, color modifiers, blockquotes, and ul tags.

Vacation: This page applies the column blog format to what we learned about bootstrap.

Wikipedia: This page was our shared attempt to recreate the wiki homepage using everything we've learned this week.

Wikipedia #2: I went home and did some additional touch ups on my own time to our wiki page. This site reflects those changes.
